O que é um script como criar um ?

UM SCRIPT É UMA SÉRIA DE INSTRUÇÕES:

Um script é uma série de instruções que um computador pode seguir para alcançar um objetivo. Você pode comparar scripts com qualquer um dos seguintes:

    1. RECEITAS
    2. GUIAS
    3. Manuais

ESCREVENDO UM SCRIPT:

Para escrever um script, primeiro você precisa indicar seu objetivo e então listar as tarefas que precisam ser concluidas para alcança-lo.

Comece com o quadro geral do que você deseja alcançar, e divida isso em passos menores.
